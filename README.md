## Installation

1. Clone the repository

   ```bash
   git clone https://github.com/bossbadi/perceptron
   ```

2. Install the dependencies

   ```bash
   npm install
   ```

3. Run the application in development mode

   ```bash
   npm run dev
   ```

   or in production mode

   ```bash
   npm run build
   npm run preview
   ```
